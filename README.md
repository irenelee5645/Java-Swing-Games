# Java-Swing-Games
Projects using Java Swing

Drawing Panel
- Analogous to MS Paint program.
- Can Save and Open files with file type .drp
- features:
  - Shapes:
     - rectangle
     - rounded rectangle
     - oval
     - PolyLine
     - Polygon
     - Pen
   - Add Ons
     - Stroke
     - Line Width
     - Fill
     - Border
     - Color selection
     - Pen Width
   - Actions
     - move
     - rotate
     - resize
     - undo/redo
